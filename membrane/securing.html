This is an extension of the Duple abstract base service that implements a dialogue sort of protection for the duplex channel. It means that a client who made the initial GET request must be ensured to be the same client that tried to connect over duplex channel. Also, each request has a TTL for the duplex connection. The implementation depends on RedisClient and running Redis!
